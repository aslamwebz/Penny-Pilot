# Pilot Your Finances

# Run Locally

# Clone the project

git clone https://github.com/aslamwebz/Penny-Pilot.git

# Go to the project directory

cd shadcn-admin

# Install dependencies

npm install

# Start the server

npm run dev
  
# What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

# Pilot Your Finances

## Run Locally

### Clone the project

```bash
git clone https://github.com/aslamwebz/Penny-Pilot.git
```

### Go to the project directory

```bash
cd Penny-Pilot
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
